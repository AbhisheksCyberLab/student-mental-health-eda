# Student Mental Health & Burnout – Exploratory Data Analysis

## 1. Overview
This project performs an in-depth Exploratory Data Analysis (EDA) on student mental health and burnout data to uncover patterns, relationships, and key risk factors influencing student well-being.

The analysis is designed as a structured data pipeline to support downstream tasks such as predictive modeling, risk assessment, and data-driven decision-making.



## 2. Objectives
- Analyze student mental health and burnout indicators  
- Identify key factors contributing to burnout and psychological stress  
- Understand relationships between academic, behavioral, and mental health variables  
- Generate actionable insights for further modeling and intervention strategies  



## 3. Dataset Description
The dataset contains features related to student lifestyle, academic workload, and mental health indicators.

Key aspects:
- Includes behavioral, academic, and psychological attributes  
- Contains both numerical and categorical variables  
- Suitable for statistical analysis and predictive modeling  



## 4. Pipeline Overview

The analysis follows a structured pipeline to ensure clarity, consistency, and reproducibility.

### 4.1 Data Ingestion & Quality Audit
- Performed schema validation and datatype checks  
- Analyzed missing values and data completeness  
- Profiled dataset structure to ensure data integrity  

### 4.2 Univariate Analysis
- Examined distributions of individual features  
- Identified skewness, outliers, and data spread  
- Evaluated feature-level characteristics  

### 4.3 Bivariate & Multivariate Analysis
- Analyzed correlations between variables  
- Explored conditional distributions and feature interactions  
- Identified relationships impacting mental health outcomes  

### 4.4 Mental Health Risk Landscape
- Segmented data into risk levels based on key indicators  
- Profiled high-risk vs low-risk student groups  
- Highlighted patterns associated with increased burnout  

### 4.5 Burnout Drivers
- Identified important features using mutual information  
- Used visual analysis to interpret feature contributions  
- Determined key drivers influencing burnout levels  

### 4.6 Key Takeaways
- Extracted actionable insights for downstream machine learning models  
- Identified high-impact variables for prediction tasks  
- Provided a foundation for building risk prediction systems  


## 5. Results and Insights
- Identified strong relationships between workload, lifestyle factors, and burnout  
- Highlighted key variables contributing to mental health risk  
- Established a structured understanding of student well-being patterns
